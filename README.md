## 📖 Overview
English Janala is an **interactive EdTech platform** designed to help users learn English vocabulary and language skills in a fun and engaging way. Built purely with vanilla JavaScript, HTML5, and Tailwind CSS — no frameworks needed!

---

## ✨ Main Features
- 📝 Interactive vocabulary learning lessons
- 🔊 Word pronunciation and definitions
- 📊 Progress tracking for learners
- 🎮 Engaging and beginner-friendly UI
- 📱 Fully responsive across all screen sizes
- ⚡ Lightweight — no frameworks, pure vanilla JS

---

## 🛠️ Tech Stack
| Category | Technology |
|----------|-----------|
| Markup | HTML5 |
| Styling | Tailwind CSS |
| Logic | Vanilla JavaScript ES6+ |
| Deployment | GitHub Pages |

---

## 📦 Dependencies
This project has **no npm dependencies** — it runs entirely in the browser with:
- Tailwind CSS (via CDN)
- Vanilla JavaScript ES6+

---

## 🚀 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/zihad-1883/Project-English-Janala.git

# 2. Go into the project directory
cd Project-English-Janala

# 3. Open index.html directly in your browser
# Simply double-click index.html OR use Live Server in VS Code
```

> 💡 No build step needed! Just open `index.html` in any browser.
